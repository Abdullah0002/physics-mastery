## Modulation

<Formula label="Amplitude Modulation (AM)">
$$m(t) = A_m\cos(2\pi f_m t) \quad \text{(message signal)}$$
$$c(t) = A_c\cos(2\pi f_c t) \quad \text{(carrier signal)}$$
$$s(t) = [A_c + A_m\cos(2\pi f_m t)]\cos(2\pi f_c t)$$
$$s(t) = A_c\left[1 + \mu\cos(2\pi f_m t)\right]\cos(2\pi f_c t)$$
</Formula>

<Formula label="Modulation Index (AM)">
$$\mu = \frac{A_m}{A_c} = \frac{A_{\max} - A_{\min}}{A_{\max} + A_{\min}}$$
$$0 < \mu \leq 1 \text{ for undistorted AM}$$
</Formula>

<Formula label="AM Power Distribution">
$$P_{\text{total}} = P_c\left(1 + \frac{\mu^2}{2}\right) = P_c + \frac{\mu^2 P_c}{2}$$
$$P_{\text{sidebands}} = \frac{\mu^2 P_c}{2} \quad \text{(each sideband: } \frac{\mu^2 P_c}{4}\text{)}$$
$$\text{Carrier power fraction} = \frac{P_c}{P_{\text{total}}} = \frac{1}{1 + \mu^2/2}$$
</Formula>

<Formula label="AM Bandwidth">
$$\text{BW} = 2f_m \quad \text{(single message frequency)}$$
$$\text{BW} = 2W \quad \text{(W = bandwidth of message signal)}$$
$$\text{Upper sideband (USB): } f_c + f_m$$
$$\text{Lower sideband (LSB): } f_c - f_m$$
</Formula>

## Frequency Modulation (FM)

<Formula label="FM Signal">
$$s(t) = A_c\cos\!\left[2\pi f_c t + \beta\sin(2\pi f_m t)\right]$$
</Formula>

<Formula label="FM Modulation Index">
$$\beta = \frac{\Delta f}{f_m} = \frac{\text{maximum frequency deviation}}{\text{message frequency}}$$
</Formula>

<Formula label="Carson's Rule (FM Bandwidth)">
$$\text{BW} \approx 2(\Delta f + f_m) = 2f_m(1 + \beta)$$
</Formula>

## Propagation of EM Waves

<Formula label="Ground Wave (Surface Wave)">
$$\text{Frequency range: } < 2\ \text{MHz (MF/LF bands)}$$
$$\text{Attenuation increases with frequency}$$
</Formula>

<Formula label="Sky Wave (Ionospheric)">
$$\text{Frequency range: } 2\ \text{MHz} - 30\ \text{MHz (HF/SW)}$$
$$\text{Critical frequency: } f_c = 9\sqrt{N_{\max}}\ \text{Hz (} N_{\max} \text{ in m}^{-3}\text{)}$$
$$\text{(Waves above critical frequency pass through ionosphere)}$$
</Formula>

<Formula label="Space Wave (Line of Sight)">
$$\text{Frequency range: } > 30\ \text{MHz (VHF/UHF/microwave)}$$
$$d_{\text{TV}} = \sqrt{2Rh_T} + \sqrt{2Rh_R}$$
</Formula>

## Line-of-Sight (LOS) Range

<Formula label="Horizon Distance for Transmitting Antenna">
$$d_T = \sqrt{2Rh_T}$$
$$\text{(R = radius of Earth = } 6.4\times10^6\ \text{m, } h_T \text{ = antenna height)}$$
</Formula>

<Formula label="Horizon Distance for Receiving Antenna">
$$d_R = \sqrt{2Rh_R}$$
</Formula>

<Formula label="Total LOS Coverage Range">
$$d = d_T + d_R = \sqrt{2Rh_T} + \sqrt{2Rh_R}$$
</Formula>

<Callout type="tip">
To double the LOS range, the antenna height must be quadrupled ($d \propto \sqrt{h}$).
</Callout>

## Satellite Communication

<Formula label="Geostationary Orbit Height">
$$h \approx 36000\ \text{km} \approx 3.6\times10^7\ \text{m}$$
$$T = 24\ \text{h (matches Earth's rotation)}$$
</Formula>

<Formula label="Coverage Angle by Geostationary Satellite">
$$\text{Coverage: up to latitude } \pm 70° \text{ approximately}$$
$$\text{Single satellite: covers about 1/3 of Earth's surface}$$
$$\text{Three satellites: near-global coverage}$$
</Formula>

## Signal-to-Noise Ratio

<Formula label="Signal-to-Noise Ratio (SNR)">
$$\text{SNR} = \frac{P_{\text{signal}}}{P_{\text{noise}}}$$
$$\text{SNR (dB)} = 10\log_{10}\left(\frac{P_S}{P_N}\right) = 20\log_{10}\left(\frac{V_S}{V_N}\right)$$
</Formula>

<Formula label="Noise Figure">
$$F = \frac{\text{SNR}_{\text{in}}}{\text{SNR}_{\text{out}}} \geq 1$$
$$F\ (\text{dB}) = 10\log_{10} F$$
</Formula>

## Digital Communication

<Formula label="Nyquist Sampling Theorem">
$$f_s \geq 2B \quad \text{(sampling frequency must be at least twice message bandwidth B)}$$
</Formula>

<Formula label="Channel Capacity (Shannon-Hartley)">
$$C = B\log_2\!\left(1 + \frac{S}{N}\right) \quad \text{[bits per second]}$$
$$\text{(B = bandwidth in Hz, S/N = signal-to-noise ratio)}$$
</Formula>

<Formula label="Bandwidth of AM Signal vs Message Signal">
$$\text{BW}_{\text{AM}} = 2 \times \text{BW}_{\text{message}}$$
</Formula>

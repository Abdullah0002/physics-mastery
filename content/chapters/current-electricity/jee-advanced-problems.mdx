## Section A: Resistance Networks & Symmetry

### Problem 1 [MCQ-M]

A cube is made of 12 identical wires, each of resistance $r$. Terminals A and B are diagonally opposite vertices of the cube. Which of the following statements are correct?

**(A)** The equivalent resistance between A and B is $\dfrac{5r}{6}$

**(B)** By symmetry, the cube can be divided into three equipotential sets: $\{A\}$, $\{3\text{ vertices adjacent to }A\}$, $\{3\text{ vertices adjacent to }B\}$, $\{B\}$

**(C)** The current through each wire adjacent to A is $\dfrac{I}{3}$, where $I$ is the total current

**(D)** If each wire has resistance $2\,\Omega$, the equivalent resistance is $\dfrac{5}{3}\,\Omega$

<details>
<summary>Solution</summary>

**Symmetry Analysis:**

Label the cube vertices. Let A be the source terminal. By symmetry:
- The 3 vertices directly connected to A are at the same potential $V_1$
- The 3 vertices directly connected to B are at the same potential $V_2$

This creates 4 equipotential groups: $\{A\}$, $\{P, Q, R\}$, $\{S, T, U\}$, $\{B\}$

**Equivalent circuit:**

- From A: 3 wires in parallel → resistance $r/3$
- From $\{P,Q,R\}$ to $\{S,T,U\}$: 6 wires, each connecting one from the first set to one from the second set → 6 parallel wires of resistance $r$ → $r/6$
- From $\{S,T,U\}$ to B: 3 wires in parallel → resistance $r/3$

$$R_{AB} = \frac{r}{3} + \frac{r}{6} + \frac{r}{3} = \frac{2r}{6} + \frac{r}{6} + \frac{2r}{6} = \frac{5r}{6}$$

**Current distribution:**

Total current $I$ splits equally at A: each of the 3 wires carries $I/3$ ✓

For $r = 2\,\Omega$: $R_{AB} = 5(2)/6 = 5/3\,\Omega$ ✓

**Answers: (A), (B), (C), (D)** — all correct.

</details>

### Problem 2 [MCQ-S]

In the infinite ladder network shown, each horizontal resistor has resistance $R$ and each vertical (shunt) resistor has resistance $R$. The equivalent resistance looking into the ladder from the left is:

$$\text{(Network: } R\text{-}R\text{-}R\text{-}\cdots \text{ with shunt } R \text{ at each node)}$$

**(A)** $R(\sqrt{3}-1)$

**(B)** $\dfrac{R(\sqrt{5}-1)}{2}$

**(C)** $R(\sqrt{2}-1)$

**(D)** $\dfrac{R(\sqrt{3}+1)}{2}$

<details>
<summary>Solution</summary>

Let the input impedance be $Z$. Since the ladder is infinite, removing one section from the left leaves the same ladder. Therefore:

$$Z = R + \frac{R \cdot Z}{R + Z}$$

Multiplying both sides by $(R + Z)$:

$$Z(R + Z) = R(R + Z) + RZ$$

$$ZR + Z^2 = R^2 + RZ + RZ$$

$$Z^2 = R^2 + RZ$$

$$Z^2 - RZ - R^2 = 0$$

Using the quadratic formula:

$$Z = \frac{R \pm \sqrt{R^2 + 4R^2}}{2} = \frac{R \pm R\sqrt{5}}{2} = \frac{R(1 \pm \sqrt{5})}{2}$$

Taking the positive root (resistance must be positive):

$$\boxed{Z = \frac{R(\sqrt{5}-1)}{2}}$$

**Answer: (B)**

</details>

### Problem 3 [INT]

In the circuit shown, five identical resistors each of resistance $R = 6\,\Omega$ are connected as follows: $R_{AB}$, $R_{BC}$, $R_{CD}$ in series forming the top path, and $R_{AD}$ directly connecting A to D with $R_{BD}$ connecting B to D. A battery of EMF $12\,\text{V}$ and negligible internal resistance is connected between A and D.

Find the power (in watts) dissipated in the resistor $R_{BD}$.

<details>
<summary>Solution</summary>

**Circuit setup:** Let the nodes be A, B, C, D with the battery between A (positive) and D (negative).

Assign node voltages: $V_A = 12\,\text{V}$, $V_D = 0$

Resistors: $R_{AB} = R_{BC} = R_{CD} = R_{AD} = R_{BD} = 6\,\Omega$

**Node B:** Connected to A via $R_{AB}$, to C via $R_{BC}$, to D via $R_{BD}$

**Node C:** Connected to B via $R_{BC}$, to D via $R_{CD}$

**KCL at node C:**
$$\frac{V_B - V_C}{R} = \frac{V_C - V_D}{R}$$
$$V_B - V_C = V_C \implies V_C = \frac{V_B}{2}$$

**KCL at node B:**
$$\frac{V_A - V_B}{R} = \frac{V_B - V_C}{R} + \frac{V_B - V_D}{R}$$

$$12 - V_B = V_B - \frac{V_B}{2} + V_B$$

$$12 - V_B = \frac{V_B}{2} + V_B = \frac{3V_B}{2}$$

$$12 = V_B + \frac{3V_B}{2} = \frac{5V_B}{2}$$

$$V_B = \frac{24}{5} = 4.8\,\text{V}$$

**Power in $R_{BD}$:**

$$P_{BD} = \frac{V_B^2}{R} = \frac{(4.8)^2}{6} = \frac{23.04}{6} = 3.84\,\text{W}$$

$$\boxed{P_{BD} \approx 3.84\,\text{W}}$$

*(Integer answer: 4 W if rounded, but exact: 3.84 W — the integer form of this problem would typically use $R = 5\,\Omega$ to give a clean answer)*

</details>

### Problem 4 [MCQ-M]

For the Wheatstone bridge shown: $P = 10\,\Omega$, $Q = 10\,\Omega$, $R = 10\,\Omega$, $S = 12\,\Omega$ (unbalanced). The galvanometer has resistance $G = 20\,\Omega$. Battery EMF $= 6\,\text{V}$, internal resistance negligible. Which of the following are correct?

**(A)** The bridge is unbalanced because $P/Q \neq R/S$

**(B)** The galvanometer current can be found using Thevenin's theorem by open-circuiting G

**(C)** The Thevenin voltage across the galvanometer terminals is $\dfrac{6}{22} - \dfrac{6}{20} \approx$ a nonzero value... [see actual calculation below]

**(D)** $I_G = \dfrac{6}{660}\,\text{A} = \dfrac{1}{110}\,\text{A}$

<details>
<summary>Solution</summary>

**Check balance:** $P/Q = 10/10 = 1$; $R/S = 10/12 \neq 1$. Unbalanced. ✓ **(A) correct**

**Thevenin across galvanometer terminals (C and D):**

Open-circuit G. The circuit becomes two voltage dividers:
- Left arm: P and R in series ($10 + 10 = 20\,\Omega$)
- Right arm: Q and S in series ($10 + 12 = 22\,\Omega$)

Battery voltage = 6 V applied across both arms (A to B).

$$V_C = V_A - I_{left} \cdot P = V_A - \frac{6}{20} \times 10 = V_A - 3\,\text{V}$$

$$V_D = V_A - I_{right} \cdot Q = V_A - \frac{6}{22} \times 10 = V_A - \frac{60}{22}\,\text{V}$$

$$V_{Th} = V_C - V_D = \frac{60}{22} - 3 = \frac{60 - 66}{22} = -\frac{6}{22} = -\frac{3}{11}\,\text{V}$$

$|V_{Th}| = 3/11\,\text{V}$. **(B) correct**

**Thevenin resistance (short battery):**

$R_{Th} = P \| R + Q \| S = (10 \| 10) + (10 \| 12)$

$= 5 + \dfrac{10 \times 12}{22} = 5 + \dfrac{120}{22} = 5 + \dfrac{60}{11} = \dfrac{55 + 60}{11} = \dfrac{115}{11}\,\Omega$

**Galvanometer current:**

$$I_G = \frac{|V_{Th}|}{R_{Th} + G} = \frac{3/11}{115/11 + 20} = \frac{3/11}{(115 + 220)/11} = \frac{3}{335} = \frac{3}{335}\,\text{A}$$

This does not equal $1/110$, so **(D) is incorrect**.

**Answers: (A), (B)**

</details>

## Section B: Multi-Loop Circuits & Kirchhoff's Laws

### Problem 5 [INT]

In the circuit: Battery $\mathcal{E}_1 = 12\,\text{V}$, $r_1 = 1\,\Omega$; Battery $\mathcal{E}_2 = 8\,\text{V}$, $r_2 = 1\,\Omega$; Battery $\mathcal{E}_3 = 4\,\text{V}$, $r_3 = 2\,\Omega$. Resistors $R_1 = 3\,\Omega$, $R_2 = 2\,\Omega$, $R_3 = 4\,\Omega$ are in three branches connecting nodes A and B. $\mathcal{E}_1$ is in branch 1, $\mathcal{E}_2$ in branch 2, $\mathcal{E}_3$ in branch 3. Find the current (in amperes, to 2 decimal places) through $R_2$.

<details>
<summary>Solution</summary>

**Apply the supernode (node voltage) method.**

Let $V_A - V_B = V$ (the node voltage difference).

For each branch, the current from A to B is (using sign convention: EMF aids current if oriented A→B):

$$I_1 = \frac{V - \mathcal{E}_1}{r_1 + R_1} = \frac{V - 12}{4}$$

$$I_2 = \frac{V - \mathcal{E}_2}{r_2 + R_2} = \frac{V - 8}{3}$$

$$I_3 = \frac{V - \mathcal{E}_3}{r_3 + R_3} = \frac{V - 4}{6}$$

**KCL at node A** (sum of currents leaving A through all branches = 0, since there's no external load):

$$I_1 + I_2 + I_3 = 0$$

$$\frac{V-12}{4} + \frac{V-8}{3} + \frac{V-4}{6} = 0$$

Multiply through by LCM = 12:

$$3(V-12) + 4(V-8) + 2(V-4) = 0$$

$$3V - 36 + 4V - 32 + 2V - 8 = 0$$

$$9V = 76$$

$$V = \frac{76}{9} \approx 8.44\,\text{V}$$

**Current through $R_2$ (branch 2):**

$$I_2 = \frac{76/9 - 8}{3} = \frac{76/9 - 72/9}{3} = \frac{4/9}{3} = \frac{4}{27} \approx 0.15\,\text{A}$$

$$\boxed{I_2 = \frac{4}{27} \approx 0.15\,\text{A}}$$

</details>

### Problem 6 [MCQ-M]

Two batteries ($\mathcal{E}_1 = 6\,\text{V}$, $r_1 = 2\,\Omega$) and ($\mathcal{E}_2 = 4\,\text{V}$, $r_2 = 1\,\Omega$) are connected in a loop with an external resistance $R = 3\,\Omega$. Which of the following are correct?

**(A)** The current in the circuit is $\dfrac{2}{6} = \dfrac{1}{3}\,\text{A}$

**(B)** Battery $\mathcal{E}_2$ is being charged

**(C)** The terminal voltage of $\mathcal{E}_1$ is $\dfrac{10}{3}\,\text{V}$

**(D)** Power delivered to $R$ is $\dfrac{1}{3}\,\text{W}$

<details>
<summary>Solution</summary>

**Net EMF** (assume both batteries oppose each other, $\mathcal{E}_1$ drives current clockwise, $\mathcal{E}_2$ opposes):

$$\mathcal{E}_{net} = \mathcal{E}_1 - \mathcal{E}_2 = 6 - 4 = 2\,\text{V}$$

**Total resistance:**

$$R_{total} = r_1 + r_2 + R = 2 + 1 + 3 = 6\,\Omega$$

**Current:**

$$I = \frac{2}{6} = \frac{1}{3}\,\text{A} \quad \checkmark \text{ (A) correct}$$

**Battery $\mathcal{E}_2$:** Current enters its positive terminal → it is being charged. **(B) correct**

**Terminal voltage of $\mathcal{E}_1$** (current leaves positive terminal → discharging):

$$V_1 = \mathcal{E}_1 - Ir_1 = 6 - \frac{1}{3}(2) = 6 - \frac{2}{3} = \frac{16}{3}\,\text{V}$$

This is NOT $10/3$. **(C) incorrect**

**Power in $R$:**

$$P_R = I^2 R = \left(\frac{1}{3}\right)^2 \times 3 = \frac{1}{9} \times 3 = \frac{1}{3}\,\text{W} \quad \checkmark \text{ (D) correct}$$

**Answers: (A), (B), (D)**

</details>

### Problem 7 [MCQ-S]

In the circuit, a battery of EMF $\mathcal{E}$ and internal resistance $r$ is connected to a network. The network consists of three resistors: $R_1$ between nodes A and B, $R_2$ between B and C, $R_3$ between A and C. The battery is connected between A and C. Find the current through $R_2$ in terms of $\mathcal{E}$, $r$, $R_1$, $R_2$, $R_3$.

**(A)** $\dfrac{\mathcal{E} R_3}{r(R_1 + R_2 + R_3) + R_1R_2}$

**(B)** $\dfrac{\mathcal{E} R_3}{r(R_1 + R_2 + R_3) + R_2R_3 + R_3R_1}$

**(C)** $\dfrac{\mathcal{E}(R_1 + R_3)}{r(R_1 + R_2 + R_3) + R_1R_2 + R_2R_3 + R_3R_1}$

**(D)** $\dfrac{\mathcal{E}}{r + R_1 \| R_2 + R_3}$

<details>
<summary>Solution</summary>

**Node voltage method:** Set $V_C = 0$, $V_A = \mathcal{E} - Ir$ (need to find).

Alternatively, treat the network as $R_1$ in series with ($R_2 \| R_3$)... but $R_3$ is directly between A and C — it's not that simple since $R_1$ and $R_3$ share node A.

**Correct approach:** $R_1$ connects A-B, $R_3$ connects A-C, $R_2$ connects B-C. Battery between A and C.

Let $V_A - V_C = V$ (terminal voltage of battery).

KCL at node B:

$$\frac{V_A - V_B}{R_1} = \frac{V_B - V_C}{R_2}$$

$$\frac{V - V_B}{R_1} = \frac{V_B}{R_2}$$

$$V R_2 = V_B(R_1 + R_2) \implies V_B = \frac{V R_2}{R_1 + R_2}$$

Current through $R_2$:

$$I_2 = \frac{V_B}{R_2} = \frac{V}{R_1 + R_2}$$

Current through $R_3$:

$$I_3 = \frac{V}{R_3}$$

Total current from battery:

$$I = I_2 + I_3 = V\left(\frac{1}{R_1 + R_2} + \frac{1}{R_3}\right)$$

Also $V = \mathcal{E} - Ir$, so:

$$I = (\mathcal{E} - Ir)\left(\frac{R_3 + R_1 + R_2}{R_3(R_1+R_2)}\right)$$

Let $D = R_3(R_1+R_2)$, $N = R_1 + R_2 + R_3$:

$$I \cdot D = \mathcal{E} N - IrN$$

$$I(D + rN) = \mathcal{E} N$$

$$I = \frac{\mathcal{E} N}{D + rN} = \frac{\mathcal{E}(R_1+R_2+R_3)}{R_3(R_1+R_2) + r(R_1+R_2+R_3)}$$

Now $V = \mathcal{E} - Ir$:

$$V = \mathcal{E}\left(1 - \frac{rN}{D + rN}\right) = \frac{\mathcal{E} D}{D + rN}$$

Current through $R_2$:

$$I_2 = \frac{V}{R_1 + R_2} = \frac{\mathcal{E} D}{(D + rN)(R_1+R_2)} = \frac{\mathcal{E} R_3(R_1+R_2)}{(R_1+R_2)[R_3(R_1+R_2) + r(R_1+R_2+R_3)]}$$

$$= \frac{\mathcal{E} R_3}{R_3(R_1+R_2) + r(R_1+R_2+R_3)}$$

Expanding the denominator:

$$= \frac{\mathcal{E} R_3}{R_1R_3 + R_2R_3 + r R_1 + r R_2 + r R_3}$$

$$= \frac{\mathcal{E} R_3}{r(R_1+R_2+R_3) + R_1R_3 + R_2R_3}$$

$$I_2 = \frac{\mathcal{E} R_3}{r(R_1+R_2+R_3) + R_2R_3 + R_3R_1}$$

There is no $R_1R_2$ term, because $R_1$ and $R_2$ form a series branch ($R_1 + R_2$) in parallel with $R_3$. This matches option (B).

**Answer: (B)**

</details>

## Section C: Star-Delta & Thevenin Transforms

### Problem 8 [MCQ-S]

Three resistors $R_1 = 6\,\Omega$, $R_2 = 3\,\Omega$, $R_3 = 9\,\Omega$ are connected in a delta (triangle) configuration between nodes A, B, C. The equivalent star (Y) resistors between the same nodes are:

**(A)** $R_A = 3\,\Omega$, $R_B = 1\,\Omega$, $R_C = 1.5\,\Omega$

**(B)** $R_A = 3\,\Omega$, $R_B = 1.5\,\Omega$, $R_C = 4.5\,\Omega$

**(C)** $R_A = 2\,\Omega$, $R_B = 1\,\Omega$, $R_C = 3\,\Omega$

**(D)** $R_A = 1.5\,\Omega$, $R_B = 3\,\Omega$, $R_C = 4.5\,\Omega$

<details>
<summary>Solution</summary>

Delta-to-Star conversion formula: If $R_{AB}$, $R_{BC}$, $R_{CA}$ are delta resistors, then:

$$R_A = \frac{R_{AB} \cdot R_{CA}}{R_{AB} + R_{BC} + R_{CA}}$$

$$R_B = \frac{R_{AB} \cdot R_{BC}}{R_{AB} + R_{BC} + R_{CA}}$$

$$R_C = \frac{R_{BC} \cdot R_{CA}}{R_{AB} + R_{BC} + R_{CA}}$$

Let $R_{AB} = R_1 = 6\,\Omega$ (between A and B), $R_{BC} = R_2 = 3\,\Omega$, $R_{CA} = R_3 = 9\,\Omega$.

Sum $= 6 + 3 + 9 = 18\,\Omega$

$$R_A = \frac{6 \times 9}{18} = \frac{54}{18} = 3\,\Omega$$

$$R_B = \frac{6 \times 3}{18} = \frac{18}{18} = 1\,\Omega$$

$$R_C = \frac{3 \times 9}{18} = \frac{27}{18} = 1.5\,\Omega$$

So: $R_A = 3\,\Omega$, $R_B = 1\,\Omega$, $R_C = 1.5\,\Omega$

So $R_A = 3\,\Omega$, $R_B = 1\,\Omega$, $R_C = 1.5\,\Omega$.

**Answer: (A)**

</details>

### Problem 9 [INT]

A Wheatstone bridge has $P = 100\,\Omega$, $Q = 100\,\Omega$, $R = 100\,\Omega$, $S = 102\,\Omega$. A galvanometer of resistance $G = 10\,\Omega$ is connected across the bridge. Battery EMF $= 5\,\text{V}$, internal resistance $= 1\,\Omega$.

Using Thevenin's theorem, find the galvanometer current in microamperes (µA). Express your answer as an integer.

<details>
<summary>Solution</summary>

**Step 1: Open-circuit voltage (Thevenin voltage)**

Remove galvanometer. Two voltage dividers across the battery terminal voltage:

With $r = 1\,\Omega$ and the two divider arms in parallel:

Left arm: $P + R = 200\,\Omega$; Right arm: $Q + S = 202\,\Omega$

These arms are in parallel: $R_{parallel} = \dfrac{200 \times 202}{200 + 202} = \dfrac{40400}{402} \approx 100.5\,\Omega$

Total resistance: $1 + 100.5 = 101.5\,\Omega$

Terminal voltage: $V_{terminal} = 5 \times \dfrac{100.5}{101.5} \approx 4.951\,\text{V}$

More precisely, voltage across the network: $V = 5 - I_{total} \times 1$

$I_{total} = \dfrac{5}{1 + 40400/402} = \dfrac{5 \times 402}{402 + 40400} = \dfrac{2010}{40802} \approx 0.04926\,\text{A}$

$V = 5 - 0.04926 \approx 4.9507\,\text{V}$

Node voltages (relative to negative terminal):

$$V_C = V \cdot \frac{R}{P+R} = 4.9507 \times \frac{100}{200} = 2.4754\,\text{V}$$

$$V_D = V \cdot \frac{S}{Q+S} = 4.9507 \times \frac{102}{202} = 4.9507 \times 0.5050 = 2.5001\,\text{V}$$

$$V_{Th} = V_D - V_C = 2.5001 - 2.4754 = 0.02467\,\text{V} \approx 24.67\,\text{mV}$$

**Step 2: Thevenin resistance (short battery, keep $r$)**

For the Thevenin resistance, short the battery (leaving $r = 1\,\Omega$ in place) and look into the galvanometer terminals C–D.

**Simplification for this problem level:** Since $r = 1\,\Omega \ll 100\,\Omega$, use $r \approx 0$:

$$R_{Th} \approx (P \| Q) + (R \| S) = \frac{100 \times 100}{200} + \frac{100 \times 102}{202} = 50 + 50.495 = 100.495\,\Omega$$

$$I_G = \frac{V_{Th}}{R_{Th} + G} = \frac{0.02467}{100.495 + 10} = \frac{0.02467}{110.495} \approx 2.23 \times 10^{-4}\,\text{A} \approx 0.223\,\text{mA}$$

**Answer: 223 µA** (i.e. $I_G \approx 0.223$ mA).

</details>

## Section D: EMF, Internal Resistance & Potentiometer

### Problem 10 [MCQ-M]

A potentiometer wire of length $L = 1\,\text{m}$ has a total resistance of $R_0 = 10\,\Omega$. A driver battery of EMF $\mathcal{E}_0 = 2\,\text{V}$ and internal resistance $r_0 = 1\,\Omega$ is connected across the wire. An unknown EMF source $\mathcal{E}$ with internal resistance $r$ gives null deflection at length $l = 60\,\text{cm}$. 

Which of the following are correct?

**(A)** The potential gradient along the wire is $\dfrac{20}{11}\,\text{V/m}$

**(B)** $\mathcal{E} = \dfrac{12}{11}\,\text{V}$

**(C)** The sensitivity of the potentiometer is independent of $r$

**(D)** If $r_0$ increases, the balance length increases for the same $\mathcal{E}$

<details>
<summary>Solution</summary>

**Potential gradient:**

$$k = \frac{\mathcal{E}_0}{r_0 + R_0} \times \frac{R_0}{L} = \frac{2}{1 + 10} \times \frac{10}{1} = \frac{20}{11}\,\text{V/m} \quad \checkmark \text{ (A) correct}$$

**Unknown EMF:**

$$\mathcal{E} = k \cdot l = \frac{20}{11} \times 0.60 = \frac{12}{11}\,\text{V} \approx 1.09\,\text{V} \quad \checkmark \text{ (B) correct}$$

**Sensitivity:** At null point, no current flows through the unknown EMF branch — the internal resistance $r$ has no voltage drop. So $r$ does not affect the balance condition. $\checkmark$ **(C) correct**

**Effect of increasing $r_0$:** Higher $r_0$ → lower current → lower potential gradient → to achieve the same $\mathcal{E} = k' \cdot l'$, since $k'$ decreased, $l'$ must increase. **(D) correct** ✓

**Answers: (A), (B), (C), (D)** — all correct.

</details>

### Problem 11 [MCQ-S]

A potentiometer is used to determine the internal resistance of a cell. The cell $(\mathcal{E}, r)$ is connected to the potentiometer wire through a resistance box $R_b$. 

- Without the resistance box in circuit (open), balance length = $l_1 = 80\,\text{cm}$
- With $R_b = 40\,\Omega$ in circuit, balance length = $l_2 = 64\,\text{cm}$

Find $r$.

**(A)** $8\,\Omega$

**(B)** $10\,\Omega$

**(C)** $12\,\Omega$

**(D)** $16\,\Omega$

<details>
<summary>Solution</summary>

**Open circuit:** Balance length $l_1$ corresponds to $\mathcal{E}$ (no current through cell branch, so no drop across $r$):

$$\mathcal{E} = k \cdot l_1$$

**With $R_b$:** Current flows through the cell branch. Terminal voltage:

$$V = \mathcal{E} - Ir = \mathcal{E} \cdot \frac{R_b}{R_b + r}$$

This terminal voltage equals the potentiometer balance:

$$V = k \cdot l_2$$

Therefore:

$$\frac{V}{\mathcal{E}} = \frac{l_2}{l_1} = \frac{R_b}{R_b + r}$$

$$\frac{64}{80} = \frac{40}{40 + r}$$

$$\frac{4}{5} = \frac{40}{40 + r}$$

$$4(40 + r) = 5 \times 40 = 200$$

$$160 + 4r = 200$$

$$4r = 40 \implies r = 10\,\Omega$$

$$\boxed{r = 10\,\Omega}$$

**Answer: (B)**

</details>

### Problem 12 [INT]

Two cells with EMFs $\mathcal{E}_1 = 6\,\text{V}$ and $\mathcal{E}_2 = 4\,\text{V}$ and internal resistances $r_1 = 2\,\Omega$ and $r_2 = 1\,\Omega$ are connected in series to a resistance $R = 7\,\Omega$. Find the terminal voltage (in volts) across the second cell.

<details>
<summary>Solution</summary>

**Net EMF** (cells in series, same direction):

$$\mathcal{E}_{net} = \mathcal{E}_1 + \mathcal{E}_2 = 10\,\text{V}$$

**Total resistance:**

$$R_{total} = r_1 + r_2 + R = 2 + 1 + 7 = 10\,\Omega$$

**Current:**

$$I = \frac{10}{10} = 1\,\text{A}$$

**Terminal voltage of $\mathcal{E}_2$** (current leaves positive terminal — discharging):

$$V_2 = \mathcal{E}_2 - I \cdot r_2 = 4 - 1 \times 1 = 3\,\text{V}$$

$$\boxed{V_2 = 3\,\text{V}}$$

</details>

## Section E: RC Circuits & Transients

### Problem 13 [MCQ-M]

A capacitor $C$ is charged through resistor $R_1$ from a battery of EMF $\mathcal{E}$ and negligible internal resistance. At $t = 0$, the charging is complete ($q_0 = C\mathcal{E}$). The battery is then disconnected and the capacitor discharges through $R_2$. 

Which of the following are correct?

**(A)** The time constant for discharging is $\tau_d = R_2 C$

**(B)** The energy dissipated in $R_2$ equals $\dfrac{1}{2}C\mathcal{E}^2$

**(C)** The charge at time $t$ during discharge is $q = C\mathcal{E}\,e^{-t/R_2 C}$

**(D)** The initial discharging current is $\dfrac{\mathcal{E}}{R_1 + R_2}$

<details>
<summary>Solution</summary>

**(A)** During discharge through $R_2$ only: $\tau = R_2 C$. ✓

**(B)** All energy stored in capacitor dissipates in $R_2$ (only resistor in discharge loop):
$$U = \frac{1}{2}C\mathcal{E}^2 \quad \checkmark$$

**(C)** $q(t) = q_0 e^{-t/\tau} = C\mathcal{E}\,e^{-t/R_2 C}$ ✓

**(D)** At $t = 0$ of discharge, only $R_2$ is in circuit (battery disconnected):
$$I_0 = \frac{q_0/C}{R_2} = \frac{\mathcal{E}}{R_2}$$
NOT $\mathcal{E}/(R_1 + R_2)$. **(D) incorrect**

**Answers: (A), (B), (C)**

</details>

### Problem 14 [MCQ-S]

In a circuit, a resistor $R = 2\,\text{M}\Omega$ and capacitor $C = 1\,\mu\text{F}$ are in series with a battery of EMF $\mathcal{E} = 10\,\text{V}$. At $t = 0$, switch is closed (capacitor initially uncharged). Find the charge on the capacitor at $t = \tau = RC$.

**(A)** $C\mathcal{E}(1 - 1/e) \approx 6.32\,\mu\text{C}$

During charging:

### Problem 15 [MCQ-M]

An RC circuit has two capacitors $C_1 = 2\,\mu\text{F}$ and $C_2 = 3\,\mu\text{F}$ connected in series, and a resistor $R = 100\,\text{k}\Omega$ in series with the combination. A battery of $10\,\text{V}$ charges the system. Which of the following are correct at steady state?

**(A)** Both capacitors carry the same charge $q = 12\,\mu\text{C}$

**(B)** Voltage across $C_1$ is $6\,\text{V}$ and across $C_2$ is $4\,\text{V}$

**(C)** The time constant for charging is $R \cdot \dfrac{C_1 C_2}{C_1 + C_2} = 120\,\text{ms}$

**(D)** At steady state, no current flows through $R$

<details>
<summary>Solution</summary>

**Equivalent capacitance:**

$$C_{eq} = \frac{C_1 C_2}{C_1 + C_2} = \frac{2 \times 3}{5} = 1.2\,\mu\text{F}$$

**Steady-state charge:**

$$q = C_{eq} \times V = 1.2 \times 10 = 12\,\mu\text{C} \quad \checkmark \text{ (A) correct}$$

**Voltages at steady state:**

$$V_1 = \frac{q}{C_1} = \frac{12}{2} = 6\,\text{V}; \quad V_2 = \frac{q}{C_2} = \frac{12}{3} = 4\,\text{V} \quad \checkmark \text{ (B) correct}$$

**Time constant:** $\tau = R C_{eq} = 10^5 \times 1.2 \times 10^{-6} = 0.12\,\text{s} = 120\,\text{ms}$ ✓ **(C) correct**

**Steady state:** Capacitors fully charged → no current through circuit ✓ **(D) correct**

**Answers: (A), (B), (C), (D)** — all correct.

</details>

In an RC circuit, the capacitor ($C = 2\,\mu\text{F}$, initially uncharged) is charged through two resistors $R_1 = 1\,\text{M}\Omega$ (in series with battery) and $R_2 = 1\,\text{M}\Omega$ (in parallel with capacitor). Battery EMF $= 12\,\text{V}$.

Find the steady-state voltage (in volts) across the capacitor.

### Problem 17 [MCQ-M]

A battery of EMF $\mathcal{E}$ and internal resistance $r$ is connected to an external resistance $R$. Which of the following are correct?

**(A)** Power delivered to external circuit is maximum when $R = r$

**(B)** At maximum power transfer, efficiency is exactly 50%

**(C)** The power dissipated in $r$ always equals the power delivered to $R$ at maximum power condition

**(D)** Maximum power delivered to $R$ is $\dfrac{\mathcal{E}^2}{4r}$

<details>
<summary>Solution</summary>

Power in $R$:

$$P_R = I^2 R = \frac{\mathcal{E}^2 R}{(R+r)^2}$$

For maximum, $\dfrac{dP_R}{dR} = 0$:

$$\frac{d}{dR}\left[\frac{R}{(R+r)^2}\right] = \frac{(R+r)^2 - R \cdot 2(R+r)}{(R+r)^4} = \frac{(R+r) - 2R}{(R+r)^3} = \frac{r-R}{(R+r)^3} = 0$$

$$\implies R = r \quad \checkmark \text{ (A)}$$

At $R = r$: $P_r = I^2 r = P_R$ (since both have same $R = r$) ✓ **(C) correct**

Total power = $\mathcal{E} I = \mathcal{E}^2/(2r)$; power to $R = \mathcal{E}^2/4r$.

Efficiency = $P_R / P_{total} = (\mathcal{E}^2/4r) / (\mathcal{E}^2/2r) = 1/2 = 50\%$ ✓ **(B) correct**

$$P_{R,max} = \frac{\mathcal{E}^2}{4r} \quad \checkmark \text{ (D) correct}$$

**Answers: (A), (B), (C), (D)** — all correct.

</details>

## Section G: Complex Network Problems

### Problem 20 [MCQ-S]

In the circuit, a battery of EMF $10\,\text{V}$ and internal resistance $1\,\Omega$ is connected to the network: Nodes A, B, C, D form a square. $R_{AB} = R_{CD} = 2\,\Omega$ (top and bottom), $R_{BC} = R_{AD} = 3\,\Omega$ (sides), $R_{AC} = 6\,\Omega$ (diagonal). Battery is connected between A and C. Find $I_{AC}$ (current through the diagonal).

**(A)** $\dfrac{10}{7}\,\text{A}$

**(B)** $1\,\text{A}$

**(C)** $\dfrac{5}{7}\,\text{A}$

**(D)** $\dfrac{50}{47}\,\text{A}$

<details>
<summary>Solution</summary>

Let node voltages: $V_A = V$ (to be found), $V_C = 0$.

**KCL at node B** (connected to A via $R_{AB} = 2\,\Omega$, to C via $R_{BC} = 3\,\Omega$):

$$\frac{V - V_B}{2} = \frac{V_B}{3}$$

$$3(V - V_B) = 2V_B$$

$$3V = 5V_B \implies V_B = \frac{3V}{5}$$

**KCL at node D** (connected to A via $R_{AD} = 3\,\Omega$, to C via $R_{CD} = 2\,\Omega$):

$$\frac{V - V_D}{3} = \frac{V_D}{2}$$

$$2(V - V_D) = 3V_D$$

$$2V = 5V_D \implies V_D = \frac{2V}{5}$$

**Total current from battery** (enters at A, leaves at C):

Current from A:
- Through $R_{AB}$: $\dfrac{V - V_B}{2} = \dfrac{V - 3V/5}{2} = \dfrac{2V/5}{2} = \dfrac{V}{5}$
- Through $R_{AD}$: $\dfrac{V - V_D}{3} = \dfrac{V - 2V/5}{3} = \dfrac{3V/5}{3} = \dfrac{V}{5}$
- Through $R_{AC}$: $\dfrac{V}{6}$

$$I_{total} = \frac{V}{5} + \frac{V}{5} + \frac{V}{6} = \frac{2V}{5} + \frac{V}{6} = \frac{12V + 5V}{30} = \frac{17V}{30}$$

But $V = \mathcal{E} - I_{total} \times r = 10 - I_{total} \times 1$:

$$I_{total} = \frac{17V}{30} = \frac{17(10 - I_{total})}{30}$$

$$30 I_{total} = 170 - 17 I_{total}$$

$$47 I_{total} = 170$$

$$I_{total} = \frac{170}{47}\,\text{A}; \quad V = 10 - \frac{170}{47} = \frac{470 - 170}{47} = \frac{300}{47}\,\text{V}$$

**Current through diagonal $R_{AC}$:**

$$I_{AC} = \frac{V}{6} = \frac{300}{47 \times 6} = \frac{300}{282} = \frac{50}{47}\,\text{A}$$

**Answer: (D)** $\dfrac{50}{47}\,\text{A} \approx 1.06\,\text{A}$.

</details>

### Problem 24 [MCQ-M]

A copper conductor of cross-sectional area $A = 1\,\text{mm}^2$ carries a current $I = 1\,\text{A}$. Given: density of copper $= 9 \times 10^3\,\text{kg/m}^3$, atomic mass $= 64\,\text{g/mol}$, free electrons per atom $= 1$, $e = 1.6 \times 10^{-19}\,\text{C}$, $N_A = 6 \times 10^{23}$.

**Answers: (A), (B), (C), (D)** — all correct.

## Quick Reference: Solution Strategies

| Problem Type | Key Technique |
|---|---|
| Symmetric network (cube, diamond) | Identify equipotential nodes → collapse → series-parallel |
| Infinite ladder | $Z = $ series element $+$ $Z \ \vert $ shunt → quadratic in $Z$ |
| Unbalanced Wheatstone | Thevenin: open G, find $V_{Th}$ and $R_{Th}$ |
| Multi-battery circuit | Node voltage method or mesh analysis |
| Potentiometer | $k = \mathcal{E}_0/(r_0 + R_0)$; null means no current, no drop across $r$ |
| RC transient | $q(t) = C V_{Th}(1 - e^{-t/R_{Th}C})$ where $V_{Th}$, $R_{Th}$ are Thevenin parameters |
| Star-Delta | $\Delta \to Y$: $R_{star} = \frac{\text{product of adjacent delta}}{\text{sum of all delta}}$ |
| Power optimization | $P_{load,max}$ when $R_{load} = R_{Th}$, efficiency $= 50\%$ |
